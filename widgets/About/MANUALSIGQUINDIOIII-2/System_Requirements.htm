System requirements

This page is based on the Reference topic template shown below - use that template to create other similar topics.

 

In order to connect to your MyCompany Player, your computer system must meet the following requirements:

PC

  • P555 or higher with 128 MB RAM

  • 49 MB free hard drive space

  • USB port

  • Internet connection (recommended)
